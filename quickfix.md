#quickfix 使用

* nmap <F6> :cn<cr>  #下一个错误
* nmap <F7> :cp<cr>  #上一项错误
* :cw 有错误打开quickfix

```bash
:cc                显示详细错误信息 ( :help :cc )
:cp                跳到上一个错误 ( :help :cp )
:cn                跳到下一个错误 ( :help :cn )
:cl                列出所有错误 ( :help :cl )
:cw                如果有错误列表，则打开quickfix窗口 ( :help :cw )
:col               到前一个旧的错误列表 ( :help :col )
:cnew              到后一个较新的错误列表 ( :help :cnew ) 

```
